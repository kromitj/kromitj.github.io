Enumberables Everyone!

Enermable is a mixin that allows collection classes, like Arrays and Hashes, the ability to easily traverse, search and sort within themselfs throught the use of the mixins methods.

Whats a mixin?

Mixins are Modules that are "missed in" to a class,givin that Class access to the Modules methods. Mixins are like a power upgrades in Mario; You can give your Array fire-power, and give your Hashes Yoshi! Mixins allow for a class to inherit functionality from multiple sources; this is good becuase Ruby only allows single inheritance; mixins are a sort-of work around that.

Back To Enumberables

Enumerable offers many very useful methods These methods have differant iterations for Arrays and Hashes. like here is the Enumerable#each method being used with an Array then with a Hash:

mario_characters = [Mario, Yoshi, Bowser, Peach, Toad] character_size = { Mario: 'Medium', Yoshi: 'Medium', Bowser: 'Large', Peach: 'Medium', Toad: 'Small'} mario_characters.each { |char| puts char} Mario Yoshi Bowser Peach Toad character_size.each { |char, size| puts "#{char}: #{size}"} Mario: Medium Yoshi: Medium Bowser: Large Peach: Medium Toad: Small

The Emumerate#each method iterates through, get this, each of the elements of the Array and what-ever code you put inside the block will be executed once for every element with the current element available for use thorugh the variables inside the pipes, | |.

How About One to Inpress Your Friends?

Now, we're going to go over an Enumerator that you may never use, but who cares, it's a method. The method is Enumberable#cyle and here is its method signature:

cycle(n=nil) { |obj| block } → nil cycle(n=nil) → an_enumerator

The signatures is saying the method takes a parameter which is nil by default then inside the block is making available the current element of the Array or Hash and execute the code put inside the block. The method will do this however many times you specified as the parameter, if you give no parameter it will cycle for inifinity, where as if you put 3, it will cycle three times.
